# NoBrokerCars — Indian Automotive Marketplace

A premium, interactive car selling platform with **zero brokerage**, featuring high-value Indian automotive models and a state-of-the-art UI/UX experience.

## ✨ Key Features
- **Curated Indian Inventory**: Featuring accurate 8k photorealistic images of popular models like Mahindra Thar, Tata Safari, and Kia Seltos.
- **Aerodynamic Minimalism**: A sleek, premium design system with Glassmorphism and high-end CSS animations.
- **Firebase Integrated**: Fully connected to a real-time cloud database (Firestore) for seamless syncing of listings, users, and inquiries.
- **Dark/Light Mode**: One-touch toggle for accessibility and comfort.
- **Database Seeder**: Includes an automated script to sync local data to the cloud in seconds.
- **WhatsApp Integration**: Direct contact with sellers via WhatsApp for quick negotiation.

## 🚀 Getting Started
1. **Clone the repository**
2. **Install dependencies**: `npm install`
3. **Configure Firebase**: Update `src/lib/firebase.js` with your project keys.
4. **Run development server**: `npm run dev`
5. **Initialize Database**: Go to Admin Settings and click the 🌱 button to sync your data.

## 🛠 Tech Stack
- **Frontend**: React.js, Vite
- **Styling**: Vanilla CSS (Premium Glassmorphism)
- **Database/Auth**: Firebase Firestore & Authentication
- **Animations**: GSAP (GreenSock Animation Platform)
